# Dixiyang Spring Boot → Python 重写接口对齐文档

> 本文档将原 `/home/lijiajia/项目/Dixiyang/dixiyang-engine` Spring Boot 项目的所有接口完整重写为 Python FastAPI 实现。

---

## 一、技术栈对照

| 组件 | Spring Boot 原版 | Python 重写版 |
|------|------------------|--------------|
| Web框架 | Spring Boot 3.4.2 | FastAPI |
| 运行方式 | Maven + JDK 17 | Uvicorn + Python 3.11 |
| ORM | MyBatis Plus | SQLAlchemy 2.0 |
| 认证 | Spring Security + JWT | PyJWT + 自定义中间件 |
| 密码加密 | BCrypt | bcrypt 5.0 |
| 数据库 | MySQL 8.0+ | SQLite（开发）/ MySQL（生产） |
| 向量数据库 | Qdrant | ChromaDB |
| API文档 | Swagger | OpenAPI (FastAPI 内置) |

## 九、前后端接口差异注意事项

1. **字段命名**：Spring Boot 使用驼峰命名（`penName`、`coverUrl`），Python 使用下划线命名（`pen_name`、`cover_url`）。
   FastAPI 自动处理 snake_case 与 camelCase 的转换，对前端透明。

2. **参数验证**：Python 版使用 Pydantic 自动验证，相比 Spring Boot 的 `@Valid` 注解更简洁。

3. **分页格式**：保持一致，返回 `records`、`total`、`size`、`current`、`pages` 字段。

4. **错误处理**：使用统一 `Result` 格式，与 Spring Boot 版完全一致。
