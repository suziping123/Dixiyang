### 5.2 第一轮：续写"暴风雨夜"之后

**输入**：

```
暴风雨夜过后，林晓月的画室被泡得一塌糊涂。请续写接下来几天的剧情，着重表现：1）苏云起和江一帆同时出现在她生活里之后，三个人之间的微妙关系；2）林晓月面对两个男人的不同态度时的内心活动——一个小心翼翼地想保护她，一个平静地把她当正常人对待。结合RAG搜索相关人物关系描写的参考。
```

**预期 AI 行为**：

| 步骤                                       | 界面反馈            |
| ---------------------------------------- | --------------- |
| `get_novel_characters`                   | 获取 5 份角色卡       |
| `get_story_nodes`                        | 获取 5 个节点内容      |
| `knowledge_search("人物关系 微妙 三人关系 小说 描写")` | 检索知识库中的人物关系描写参考 |
| `knowledge_search("残疾人 人物刻画 日常 小说")`     | 检索知识库中相关描写参考    |
| 综合生成回答                                   | 流式输出续写内容        |

**预期回答方向**（片段示例）：

```
【第二天·清晨】
雨停了。阳光从云缝里漏下来，照在湿漉漉的院子里。林晓月摇着轮椅到画室门口，门推开的一瞬间，她倒吸了一口气——画架倒了，画稿散落一地，大部分都被水泡得皱巴巴的。她默默把还能抢救的画一张一张捡起来、摊平，动作很慢。

苏云起是上午九点来的，带着热豆浆和包子。他看到屋里的狼藉，二话不说就要帮忙收拾，但林晓月拦住了他：「你把手弄脏了怎么去给病人做治疗？」苏云起僵了一下，最后还是把豆浆放下走了。

江一帆是下午来的。他没敲门，直接穿过敞开的院门走了进来。他站在画室门口侧耳听了一会儿——林晓月正在用吸水纸压平画稿。他问：「需要帮忙吗？」语气平淡，像在问一件很普通的事。林晓月说：「不用。」他说：「那我在这儿坐一会儿。」然后就在门槛上坐了下来，面朝院子，闭着眼睛。他什么也没做，但林晓月忽然觉得这个房间没那么乱了。
```

## 六、编辑修正 & 重新生成（测试后台逻辑）

在任意 AI 回答上：

### 6.1 编辑修正

| 操作 | 步骤                       |
| -- | ------------------------ |
| 点击 | AI 消息上的 **✏️ 编辑回答** 按钮   |
| 修改 | 在弹窗右侧 textarea 中修改不满意的段落 |
| 保存 | 点击保存                     |

**后台行为**：

```
PUT /api/chatHistory/message/{sessionId}
→ replace_message() 备份到 originalContent
→ 写入 edits.json → {messageIndex, originalContent, editedContent, errorType}
→ 异步 extract_keypoint_async() → 修正要点存入 edits
```

### 6.2 重新生成

| 操作 | 步骤                     |
| -- | ---------------------- |
| 点击 | AI 消息上的 **🔄 重新生成** 按钮 |
| 等待 | 流式输出新回答                |

**后台行为**：

```
POST /api/chat/regenerate
→ ChatContentFileService.truncateChain() 截断到当前消息前
→ 用同样历史 + edits 重新调用 LLM
→ 新回答追加到链尾
```

### 6.3 验证 AI 学习效果

同一 session 中，编辑修正后继续对话，观察 AI 是否吸收了修正点。
