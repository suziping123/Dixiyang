## 本次完成

- 所有字段加 `Field(alias="camelCase")` + `populate_by_name=True`
- 前端 `novelId`/`characterIds`/`sessionId`/`regenerateIndex` 等自动映射

### 2. POST /chat/stream — 完整重写
- 加载链式文件历史 + edits.json 修正上下文

### 3. POST /chat/regenerate — 完整重写
- `truncate_chain(chain_dir, regenerateIndex)` 截断链
- 截断后历史 + edit context + 角色/节点上下文构建 prompt
- 流式输出，完成后保存新 assistant 回复到链文件

### 4. POST /chat/generate-title — AI 标题生成
- 读取链前 4 条消息，调用 DeepSeek 生成 3-8 字标题
- 同步非流式调用，更新 DB

### 5. 链式文件存储
- `chat_history_service.py` 改用链式文件而非单文件
- `get_session_messages` → 从链文件读取
- `batch_save` → append 到新链文件
- `edit_message` → 重写链 + 记录 edits.json
