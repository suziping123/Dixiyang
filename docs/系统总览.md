# Dixiyang 系统总览

## 系统定位

小说创作辅助平台，提供 **小说管理 → 角色/节点/时间线创作 → RAG 智能助手** 全栈 API。

---

## 一、认证 — `auth`

| 方法 | 路径 | 说明 | JWT |
|------|------|------|-----|
| POST | `/auth/login` | 用户登录（返回 token） | ❌ |
| POST | `/auth/register` | 用户注册 | ❌ |

用户注册/登录后获取 JWT Token，后续请求通过 `Authorization: Bearer <token>` 鉴权。

## 四、RAG 知识库 — `rag`

| 方法 | 路径 | 说明 | 实现 |
|------|------|------|------|
| GET | `/rag/stats` | 向量库统计（总数/模型/维度） | Python FastAPI |
| GET | `/rag/count` | 文档总数 | Python FastAPI |
| GET | `/rag/documents` | 分页浏览文档 | Python FastAPI |
| POST | `/rag/search` | 向量语义搜索（query params） | Python FastAPI |
| POST | `/rag/embed` | 单条文本向量化 | Python FastAPI |

- Python FastAPI 端（端口 8085）提供上述 5 个端点，通过 `langchain_chroma.Chroma` 操作 ChromaDB
- Java Spring 端（端口 8084）提供相同 4 个端点（不含 embed），通过 ChromaDB REST API 直连
- 前端通过 `/api/rag/*` 代理到 Python 后端（开发模式）或 Java 后端（生产）

---

## 五、RAG 助手 — `chat`

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/chat/stream` | 流式对话（SSE） |
| POST | `/chat/regenerate` | 重新生成回答（SSE） |
| POST | `/chat` | 非流式对话 |
| GET | `/chat` | 非流式对话（GET） |

### 核心机制

1. **`req.use_rag`** — 用户开关，控制是否注入角色/节点固定上下文
2. **`is_rag_enabled()`** — 后端自动检测 Chroma 是否可达，决定是否注入 `knowledge_search` 工具
3. **`knowledge_search`** — LangChain `@tool`，LLM 自主决定是否调用

### 流式输出

```
SSE data: {"type":"thinking","delta":"..."}
SSE data: {"type":"content","delta":"..."}
SSE data: {"type":"tool_start","tool":"knowledge_search"}
SSE data: {"type":"tool_result","name":"knowledge_search","result":"..."}
SSE data: {"type":"done"}
```

## 七、端口映射

| 服务 | 端口 | 用途 |
|------|------|------|
| ChromaDB | 8000 | 向量数据库 HTTP API |
| Java Spring Boot | 8084 | 业务 API（context-path: `/api`） |
| Python FastAPI | 8085 | RAG Embedding + 知识库管理 API |
| Vue Dev Server | 5173 | 前端（代理 `/api` → 8084） |
| MySQL | 3306 | 关系数据库 |

---

## 八、数据流

```
前端 (Vue 3, 5173)
  │ 代理 /api/* → 8084
  │
  ├──> Java Spring Boot (8084)
  │     ├── auth / novel / character / storyNode / timeline — MySQL CRUD
  │     ├── rag/stats / count / documents / search — ChromaDB REST (8000)
  │     └── chat/stream — AI 对话（工具调用 → Python 8085 embed）
  │
  └──> Python FastAPI (8085)
        ├── rag/stats / count / documents / search — langchain_chroma
        ├── rag/embed — DixiyangEmbeddings → Java 8084（兜底本地模型）
        └── chat/stream — AI 对话（streaming + tool loop）
```
