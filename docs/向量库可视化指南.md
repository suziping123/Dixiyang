### 2. Python 代码直接查询

```python
import chromadb
from sentence_transformers import SentenceTransformer

# 连接向量库
client = chromadb.PersistentClient(path="storage/vectordb_4060")
col = client.get_collection("dixiyang_knowledge")

# 查看总数
print(f"文档数: {col.count()}")

# 按来源统计
book_count = col.count(where={"source": "book"})
law_count = col.count(where={"source": "law"})
print(f"书籍: {book_count}, 法律: {law_count}")

# 按分类统计
for cat in ["科幻小说", "玄幻小说", "恐怖悬疑", "推理悬疑", "科普自然"]:
    c = col.count(where={"category": cat})
    print(f"  {cat}: {c}")

# 检索测试
model = SentenceTransformer("BAAI/bge-m3", device="cpu")
query_emb = model.encode(["三体讲了什么"]).tolist()
results = col.query(query_embeddings=query_emb, n_results=5, include=["documents", "metadatas", "distances"])

for i, (doc, meta, dist) in enumerate(zip(results["documents"][0], results["metadatas"][0], results["distances"][0])):
    print(f"\n[{i+1}] 相似度: {1-dist:.4f}")
    print(f"    来源: {meta.get('source')} | 书籍: {meta.get('book_title', 'N/A')}")
    print(f"    内容: {doc[:200]}...")

# 按 source 过滤检索（只搜小说）
results = col.query(
    query_embeddings=query_emb,
    n_results=5,
    where={"source": "book"},  # 只查小说
    include=["documents", "metadatas", "distances"]
)
```

## 前端可视化方案（待实现）

### 方案 A：Python Web UI（推荐）

用 Flask 或 Streamlit 做一个简单的 Web 页面，展示向量库内容。

功能：
- 首页：统计面板（总文档数、来源分布饼图、分类分布柱状图）
- 检索页：输入 query，展示 Top-K 结果（高亮相似度、来源、书籍名）
- 浏览页：按书籍/分类筛选，浏览所有 chunk
- 管理页：查看处理日志、错误记录

技术栈：Flask + Tailwind CSS + Chart.js（或 Streamlit 一站式）

### 方案 B：Java 后端 API + Vue 前端

在 dixiyang-engine 中新增接口，Vue 前端新增可视化页面。

API 设计：
```
GET  /api/rag/stats          返回向量库统计
GET  /api/rag/search?q=xxx   检索测试
GET  /api/rag/chunks?page=1  分页浏览 chunks
```

缺点：Java 端用的是 Qdrant（不是 Chroma），需要两套数据源。

### 方案 C：CLI 导出 + 静态页面

用 `visualize_rag.py --export data.json` 导出数据，用 Vue 做静态展示。
