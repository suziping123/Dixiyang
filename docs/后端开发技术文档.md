# 后端开发技术文档

## 1. 项目概述

Dixiyang Engine 是一个基于 Spring Boot 3 的后端服务，为 Dixiyang 前端应用提供 API 支持。项目采用分层架构设计，集成了 AI 能力、向量数据库和缓存系统，为小说创作提供全方位的工具支持。

### 1.1 核心功能

- **用户认证与授权**：基于 Spring Security 和 JWT 的身份认证
- **小说管理**：创建、编辑、删除小说
- **角色管理**：小说角色的 CRUD 操作
- **关系管理**：小说之间的关联关系管理
- **故事节点**：小说内容节点的管理
- **时间线**：小说时间线的管理
- **文件管理**：文件上传和管理
- **用户配置**：用户个性化配置管理
- **RAG 系统**：基于向量数据库的检索增强生成系统

**StoryNodeServiceImpl.java**：

```java
@Service
public class StoryNodeServiceImpl extends ServiceImpl<StoryNodeMapper, StoryNode> implements IStoryNodeService {

    @Autowired
    private EmbeddingService embeddingService;

    @Override
    public Page<StoryNode> getNodePage(Long novelId, Long timelineId, int page, int pageSize) {
        LambdaQueryWrapper<StoryNode> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(StoryNode::getNovelId, novelId);
        if (timelineId != null) {
            wrapper.eq(StoryNode::getTimelineId, timelineId);
        }
        wrapper.orderByDesc(StoryNode::getCreateTime);
        return this.page(new Page<>(page, pageSize), wrapper);
    }

    @Override
    @Transactional(rollbackFor = Exception.class)
    public boolean createNode(StoryNode node) {
        try {
            // 生成向量嵌入
            if (node.getContent() != null && !node.getContent().isEmpty()) {
                String vectorId = embeddingService.generateEmbedding(node.getContent());
                node.setVectorId(vectorId);
            }
            return this.save(node);
        } catch (Exception e) {
            log.error("创建节点失败: {}", e.getMessage(), e);
            throw new RuntimeException("创建节点失败: " + e.getMessage());
        }
    }

    @Override
    @Transactional(rollbackFor = Exception.class)
    public boolean updateNode(Long id, StoryNode node) {
        try {
            node.setId(id);
            // 更新向量嵌入
            if (node.getContent() != null && !node.getContent().isEmpty()) {
                String vectorId = embeddingService.generateEmbedding(node.getContent());
                node.setVectorId(vectorId);
            }
            return this.updateById(node);
        } catch (Exception e) {
            log.error("更新节点失败: {}", e.getMessage(), e);
            throw new RuntimeException("更新节点失败: " + e.getMessage());
        }
    }

    @Override
    @Transactional(rollbackFor = Exception.class)
    public boolean deleteNode(Long id) {
        try {
            // 删除向量嵌入
            StoryNode node = this.getById(id);
            if (node != null && node.getVectorId() != null) {
                embeddingService.deleteEmbedding(node.getVectorId());
            }
            return this.removeById(id);
        } catch (Exception e) {
            log.error("删除节点失败: {}", e.getMessage(), e);
            throw new RuntimeException("删除节点失败: " + e.getMessage());
        }
    }
}
```

### 6.2 EmbeddingService 实现

```java
@Service
public class EmbeddingService {

    @Autowired
    private OpenAiEmbeddingClient embeddingClient;

    @Autowired
    private VectorStore vectorStore;

    public String generateEmbedding(String content) {
        try {
            // 生成嵌入向量
            List<Double> embedding = embeddingClient.embed(content);
            
            // 存储到向量数据库
            Document document = new Document(
                content,
                Map.of(
                    "content", content,
                    "timestamp", System.currentTimeMillis()
                )
            );
            vectorStore.add(document);
            
            return document.getId();
        } catch (Exception e) {
            log.error("生成嵌入失败: {}", e.getMessage(), e);
            throw new RuntimeException("生成嵌入失败: " + e.getMessage());
        }
    }

    public void deleteEmbedding(String vectorId) {
        try {
            vectorStore.delete(vectorId);
        } catch (Exception e) {
            log.error("删除嵌入失败: {}", e.getMessage(), e);
        }
    }

    public List<Document> searchSimilar(String query, int topK) {
        try {
            return vectorStore.similaritySearch(query, topK);
        } catch (Exception e) {
            log.error("搜索相似内容失败: {}", e.getMessage(), e);
            return Collections.emptyList();
        }
    }
}
```

**ChatController.java**：

```java
@RestController
@RequestMapping("/chat")
@Tag(name = "聊天模块")
public class ChatController {

    @Autowired
    private OpenAiChatClient chatClient;

    @Autowired
    private EmbeddingService embeddingService;

    @PostMapping("/query")
    public Result<String> chat(@RequestBody Map<String, String> request) {
        String query = request.get("query");
        
        // 搜索相关内容
        List<Document> similarDocs = embeddingService.searchSimilar(query, 3);
        
        // 构建上下文
        StringBuilder context = new StringBuilder();
        context.append("相关内容:\n");
        for (Document doc : similarDocs) {
            context.append(doc.getContent()).append("\n");
        }
        context.append("\n用户问题: " + query);
        
        // 调用 AI 模型
        ChatResponse response = chatClient.call(
```

```java
@Component
public class RedisUtils {

    @Autowired
    private RedisTemplate<String, Object> redisTemplate;

    public void set(String key, Object value, long timeout, TimeUnit unit) {
        redisTemplate.opsForValue().set(key, value, timeout, unit);
    }

    public Object get(String key) {
        return redisTemplate.opsForValue().get(key);

    public void delete(String key) {
```

### 7.3 缓存使用示例

在 `NovelServiceImpl.java` 中使用缓存：

```java
@Service
public class NovelServiceImpl extends ServiceImpl<NovelMapper, Novels> implements NovelService {

    @Autowired
    private RedisUtils redisUtils;

    @Override
    public NovelVO getNovelById(Long id) {
        // 尝试从缓存获取
        String cacheKey = "novel:" + id;
        NovelVO novelVO = (NovelVO) redisUtils.get(cacheKey);
        if (novelVO != null) {
            return novelVO;
        }
        
        // 从数据库获取
        Novels novel = this.getById(id);
        if (novel == null) {
            return null;
        }
        
        // 转换为 VO
        novelVO = new NovelVO();
        BeanUtils.copyProperties(novel, novelVO);
        
        // 存入缓存
        redisUtils.set(cacheKey, novelVO, 30, TimeUnit.MINUTES);
        
        return novelVO;
    }

    @Override
    @Transactional(rollbackFor = Exception.class)
    public NovelVO updateNovel(Long userId, Long novelId, Novels novel, MultipartFile coverFile) {
        // 更新数据库
        novel.setId(novelId);
        this.updateById(novel);
        
        // 清除缓存
        String cacheKey = "novel:" + novelId;
        redisUtils.delete(cacheKey);
        
        // 返回更新后的小说
        return getNovelById(novelId);
    }
}
```

## 10. 常见问题与解决方案

### 10.1 依赖注入问题

**问题**：Service 或 Mapper 注入失败

**解决方案**：
- 确保类上有 `@Service`、`@Mapper` 等注解
- 检查包扫描路径是否正确
- 确保依赖关系正确

### 10.2 事务管理问题

**问题**：事务不回滚

**解决方案**：
- 在 Service 方法上添加 `@Transactional` 注解
- 确保异常被正确抛出
- 配置事务管理器

### 10.3 数据库连接问题

**问题**：数据库连接失败

**解决方案**：
- 检查数据库连接配置
- 确保数据库服务运行
- 检查网络连接

### 10.4 JWT 令牌问题

**问题**：令牌无效或过期

**解决方案**：
- 检查令牌生成逻辑
- 确保令牌验证逻辑正确
- 合理设置令牌过期时间

### 10.5 RAG 系统问题

**问题**：向量嵌入失败

**解决方案**：
- 检查 API 密钥是否正确
- 确保 Qdrant 服务运行
- 检查网络连接
